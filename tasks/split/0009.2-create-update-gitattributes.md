# Subtask 0009.2: Create/Update .gitattributes

**Parent Task**: 0009 - Packaging & Repository Hygiene  
**Effort**: S (1h)  
**Dependencies**: None

## Description

Create or update .gitattributes to exclude development files from distribution packages.

## Actions

- [ ] Create .gitattributes if missing
- [ ] Add export-ignore rules for tests/, docs/, .github/, examples/, benchmarks/
- [ ] Add line ending normalization (*.php text eol=lf)
- [ ] Test export works correctly (git archive)
- [ ] Verify unnecessary files excluded

## Acceptance Criteria

- .gitattributes file exists
- Export-ignore rules added for dev files
- Line ending normalization added
- Export tested
- Unnecessary files excluded from dist

## Definition of Done

- [ ] .gitattributes created/updated
- [ ] Export tested (git archive --format=zip HEAD > test.zip)
- [ ] Unnecessary files excluded from dist
- [ ] Changes committed to version control

## Tests Needed

- Test: `git archive --format=zip HEAD > test.zip` then inspect contents
- Verify tests/, .github/, etc. not in archive
