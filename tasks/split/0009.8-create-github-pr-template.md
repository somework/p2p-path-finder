# Subtask 0009.8: Create GitHub PR Template

**Parent Task**: 0009 - Packaging & Repository Hygiene  
**Effort**: S (30min)  
**Dependencies**: None

## Description

Create GitHub pull request template to guide contributors.

## Actions

- [ ] Create .github/PULL_REQUEST_TEMPLATE.md
- [ ] Include: description, related issues, checklist
- [ ] Keep it reasonable (not overwhelming)
- [ ] Include common checklist items (tests, docs, changelog)
- [ ] Test template works on GitHub

## Acceptance Criteria

- PULL_REQUEST_TEMPLATE.md created
- Includes description and checklist
- Reasonable length
- Checklist useful
- Tested on GitHub

## Definition of Done

- [ ] PR template created
- [ ] Template tested
- [ ] Checklist useful
- [ ] Not overwhelming
- [ ] Changes committed to version control

## Tests Needed

- Manual test: create test PR on GitHub and verify template appears
