# Subtask 0009.7: Create GitHub Issue Templates

**Parent Task**: 0009 - Packaging & Repository Hygiene  
**Effort**: M (1-2h)  
**Dependencies**: None

## Description

Create GitHub issue templates for bug reports and feature requests.

## Actions

- [ ] Create .github/ISSUE_TEMPLATE/bug_report.md
- [ ] Create .github/ISSUE_TEMPLATE/feature_request.md
- [ ] Make them helpful but not burdensome
- [ ] Include relevant sections (description, steps, environment, etc.)
- [ ] Test templates work on GitHub

## Acceptance Criteria

- bug_report.md template created
- feature_request.md template created
- Templates helpful
- Not burdensome
- Tested on GitHub

## Definition of Done

- [ ] Issue templates created
- [ ] Templates tested
- [ ] Helpful format
- [ ] Not too verbose
- [ ] Changes committed to version control

## Tests Needed

- Manual test: create test issue on GitHub using templates
