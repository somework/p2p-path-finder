# Subtask 0009.14: Test Package Installation

**Parent Task**: 0009 - Packaging & Repository Hygiene  
**Effort**: S (30min)  
**Dependencies**: All 0009 tasks

## Description

Test that the package can be installed and used in a fresh project.

## Actions

- [ ] Create fresh directory outside repository
- [ ] Install package: `composer require somework/p2p-path-finder:dev-main`
- [ ] Verify autoloading works
- [ ] Run simple example from docs
- [ ] Verify no errors

## Acceptance Criteria

- Fresh installation tested
- Package installs without errors
- Autoloading works
- Simple example runs
- No errors or warnings

## Definition of Done

- [ ] Installation tested
- [ ] Works without errors
- [ ] Autoloading verified
- [ ] Example runs successfully
- [ ] Package ready for distribution

## Tests Needed

Create test script in fresh directory:
```php
<?php
require 'vendor/autoload.php';
use Somework\\P2PPathFinder\\...;
// Simple example
```
